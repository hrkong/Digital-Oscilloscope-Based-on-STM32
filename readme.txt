2023山大电赛校赛题目-简易示波器
实验器材:
	正点原子探索者STM32F407开发板
	
实验目的:
	电赛题目-简易示波器
	
硬件资源:
	1.项目很简单，只用了stm32自带的ADC、TIM、DMA和开发板自带的ALIENTEK 4.3寸TFTLCD模块；
	2.原理是TIMER3控制ADC1采集输入波形的电压信息，通过转换之后在显示屏上显示；
	3.波形绘制使用DMA中断，可选画点和画线两种方法进行绘制；
	4.可显示频率、峰峰值、幅值、波形、占空比、水平垂直时基；
	5.ADC使用PA5引脚；
	
实现功能：
	见简易示波器(C题)pdf文件中的功能描述，除了双通道都实现了
